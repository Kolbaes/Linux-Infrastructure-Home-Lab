# Linux-Infrastructure-Home-Lab

This project documents my self-hosted Linux infrastructure built on repurposed hardware.
The Objective was to gain hands-on experience Linux administration, Networking, Monitoring, and System Troubleshooting while preparing for a career in IT Infrastructure and networking.

## Hardware
- CPU: Intel Pentium G3240
- RAM: DDR3 4GB
- Storage: 120GB SATA SSD

## Operating System
- Ubuntu 24.04.4 LTS

# Services
- Jellyfin - Media Streaming
- Radarr - Movie Automation
- Sonarr - TV show Automation
- Wireshark - Network Packer Captures
- Grafana - Monitoring Dashboard
- Tailscale - Secure Remote Access
- qBittorrent - Media Download Client
- Docker - Container Platform
